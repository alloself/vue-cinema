# vue-video-watcher

## Project setup
```
yarn install
create .env with variables:
1)VUE_APP_API_URL
2)VUE_APP_API_TOKEN
3)VUE_APP_IMAGE_PATH
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
